FS_GPlib - Graph Propagation Library
=====================================

``FS_GPlib`` is a flexible and scalable Python library designed for simulating a wide range of graph-based propagation processes on complex networks.
It supports classical epidemic models, opinion dynamics, and simulations on dynamic networks with Message Passing based dual-acceleration framework and distributed computing.
